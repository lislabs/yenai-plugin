# Yenai-Plugin说明

Yenai-Plugin是一个Yunzai-Bot的升级插件，提供对bot的一些便携操作。


#### 安装教程

请将Yenai-Plugin放置在Yunzai-Bot的plugins目录下，重启Yunzai-Bot后即可使用。

推荐使用git进行安装，以方便后续升级。在Yunzai根目录夹打开终端，运行

```
// 使用gitee
git clone https://gitee.com/yeyang52/yenai-plugin.git
```
