## 基础功能

|        功能        |                介绍                 |              指令              |
| :----------------: | :---------------------------------: | :----------------------------: |
|        禁言        |                                     |  #禁言<@群员\| QQ 号> <时间>   |
|        解禁        |                                     |        #解禁<@群员\|QQ>        |
|   全体禁言/解禁    |                                     |         #全体禁言/解禁         |
|     发全体通知     |                                     |         #发通知 <消息>         |
|   允许/禁止匿名    |                                     |       #<允许\|禁止>匿名        |
|       踢群员       |                                     |       #踢<@群员\|QQ 号>        |
|    获取禁言列表    |                                     |         #获取禁言列表          |
|    解除全部禁言    |                                     |         #解除全部禁言          |
| 清理多久没发言的人 |                                     | #<清理\| 查看><时间>没发言的人 |
|  清理从未发言的人  |                                     |   #<清理\|查看>从未发言的人    |
